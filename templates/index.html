Features

🔭

N-Body Physics

Leapfrog & RK4 integrators with Barnes-Hut octree optimization. Real gravitational interactions, not scripted animations.

🌟

Procedural Shaders

GPU-generated stellar surfaces with Planck blackbody colors. Every star, planet, and black hole is rendered with custom GLSL shaders.

🕳️

Black Holes

Gravitational lensing effects, accretion discs, and event horizons rendered with physically-inspired post-processing.

⚡

Web Workers

Physics runs in a separate thread so the render loop never drops frames, even with thousands of bodies.

💾

Cloud Save

Save your simulations, rewind to any snapshot, and share via tokenised URLs — no account required for guests.

📱

Zero Install

Runs on any device with a WebGL 2.0 capable browser. No downloads, no plugins, no subscriptions.
